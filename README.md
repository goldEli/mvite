# mvite

[![npm][npm-img]][npm-url]

Web dev tool via ES Module. Support React project development.

## Getting Stated

```shell
$ npm install mvite -g

$ mvite
Server start 4001 prot:  http://localhost:4000

$ mvite 8888
Server start 8888 prot:  http://localhost:8888
```

## Features

- [x] Start server
- [x] Static server
- [ ] Rewrite import
- [ ] Babel
- [ ] i18n
